## JVM metrics
- memory and buffer pool details
- GC statistics
- thread utilization
- Virtual threads statistics
- The number of classes loaded and unloaded
- JVM version information
- JIT compilation time

### Memory
- Heap and non-heap memory usage
- Memory pools (e.g. Eden, Survivor, Old Gen)
- Buffer pool usage (direct, mapped)

### Garbage Collection
- GC pause count and duration
- Time spent in GC
- Allocation and promotion rates

### Threads
- Live thread count
- Peak thread count
- Daemon vs non-daemon threads

### Virtual Threads (if enabled)
- Virtual thread count and activity

### Class Loading
- Number of loaded and unloaded classes

### JIT Compilation
- Total time spent in JIT compilation

### JVM Info
- JVM version
- Runtime and vendor information
